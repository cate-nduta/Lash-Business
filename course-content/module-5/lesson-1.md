# Lesson 5.1: Choosing Website Type (Portfolio, Business, Blog, Booking)

**Estimated Time**: 25 minutes

---

## Introduction

Before you start building your website, you need to decide what type of website you're creating. Different website types have different purposes, structures, and features. This lesson helps you understand the main website types and choose the right one for your needs.

**What You'll Learn:**
- The main types of websites
- What each type is best for
- How to choose the right type for your goals
- What features each type typically needs
- How your choice affects your website structure

---

## Why Website Type Matters

### Different Types, Different Purposes

**Think of it like choosing a vehicle:**
- **Sports car** - Fast, flashy, for showing off
- **SUV** - Practical, versatile, for families
- **Truck** - Work-focused, for specific tasks
- **Sedan** - Balanced, everyday use

**Websites are similar:**
- Each type serves a different purpose
- Each has different features
- Each appeals to different audiences
- Each requires different planning

**Choosing the right type helps you:**
- Focus on what matters
- Build the right features
- Attract the right audience
- Save time and effort

---

## The Main Website Types

### 1. Portfolio Website

**What it is:**
- Showcase your work
- Display projects, art, photography
- Personal brand presentation
- Professional showcase

**Best for:**
- Freelancers (designers, photographers, writers)
- Artists and creatives
- Professionals building their brand
- Job seekers showing their work

**Key features:**
- Gallery of work/projects
- About page
- Contact form
- Simple navigation
- Visual focus

**Example prompts for Cursor:**
```
I want to build a portfolio website for a photographer.
Pages: Home (gallery), About, Contact
Focus: Showcase photography work with large images
Style: Clean, minimal, image-focused
```

---

### 2. Business Website

**What it is:**
- Professional business presence
- Company information
- Service descriptions
- Contact and location info

**Best for:**
- Small businesses
- Local shops and services
- Companies needing online presence
- Professional services

**Key features:**
- Homepage with business info
- Services/products page
- About page
- Contact page
- Location/hours
- Testimonials

**Example prompts for Cursor:**
```
I want to build a business website for a local bakery.
Pages: Home, Menu, About, Contact, Location
Focus: Show products, prices, location, hours
Style: Warm, inviting, food-focused
```

---

### 3. Blog Website

**What it is:**
- Content-focused site
- Regular posts/articles
- Reader engagement
- Information sharing

**Best for:**
- Writers and bloggers
- Content creators
- Educators sharing knowledge
- Personal journals
- News sites

**Key features:**
- Blog post listings
- Individual post pages
- Categories/tags
- Search functionality
- Author info
- Comment system (optional)

**Example prompts for Cursor:**
```
I want to build a blog website about travel.
Pages: Home (blog list), About, Individual post pages
Focus: Easy reading, article navigation, categories
Style: Clean, readable, content-first
```

---

### 4. Booking Website

**What it is:**
- Appointment scheduling
- Service booking
- Calendar integration
- Payment processing

**Best for:**
- Service businesses (salons, spas, consultants)
- Appointment-based services
- Event bookings
- Class registrations

**Key features:**
- Service listings
- Booking calendar
- Time slot selection
- Payment integration
- Confirmation system
- Email notifications

**Example prompts for Cursor:**
```
I want to build a booking website for a lash studio.
Pages: Home, Services, Booking, Contact
Focus: Easy booking flow, calendar, payment
Style: Professional, clean, booking-focused
```

## How to Choose Your Website Type

### Ask Yourself These Questions

**1. What's your main goal?**
- Showcase work? → Portfolio
- Sell services? → Business or Booking
- Share content? → Blog
- Take appointments? → Booking

**2. What do visitors need to do?**
- View your work? → Portfolio
- Learn about your business? → Business
- Read articles? → Blog
- Book appointments? → Booking

**3. What's your primary content?**
- Images/projects? → Portfolio
- Business info? → Business
- Articles/posts? → Blog
- Services/appointments? → Booking

**4. How often will you update?**
- Occasionally (new projects)? → Portfolio
- Rarely (stable info)? → Business
- Regularly (new posts)? → Blog
- Frequently (availability)? → Booking

---

## Combining Types

### You Can Mix Types!

**Many websites combine types:**
- **Business + Blog** - Business site with blog section
- **Portfolio + Booking** - Showcase work and take bookings
- **Business + Portfolio** - Show services and past work

**Example:**
```
I want to build a website for a freelance designer.
Main type: Portfolio (showcase work)
Additional: Booking (take client consultations)
Pages: Home, Portfolio, Services, Booking, About, Contact
```

**When combining:**
- Choose one primary type
- Add secondary features
- Keep focus clear
- Don't try to do everything

## Making Your Decision

### Decision Framework

**Step 1: Identify your primary goal**
- Write down your main objective
- Be specific and clear

**Step 2: List what visitors need to do**
- What actions should they take?
- What information do they need?

**Step 3: Choose your primary type**
- Match your goal to a type
- Don't overthink it

**Step 4: Consider secondary features**
- What else might you need?
- Can you add it later?

**Step 5: Write your Cursor prompt**
- Include the website type
- Describe your goals
- List needed pages

---

## Common Mistakes to Avoid

### 1. Trying to Do Everything

**Bad:**
```
I want a website that's a portfolio, blog, booking site, 
and online store all in one.
```

**Good:**
```
I want a portfolio website for my design work, 
with a simple contact form for inquiries.
```

### 2. Not Being Clear About Goals

**Bad:**
```
I want a website for my business.
```

**Good:**
```
I want a business website for my yoga studio.
Visitors need to see class schedules, book classes, 
and learn about the studio.
```

### 3. Choosing Based on Trends, Not Needs

**Bad:**
```
Everyone has a blog, so I need one too.
```

**Good:**
```
I want to share weekly tips about my industry, 
so a blog makes sense for my goals.
```

## Key Takeaways

1. **Website type determines structure** - Different types need different pages and features
2. **Choose based on your goals** - Not trends or what others are doing
3. **You can combine types** - But keep one primary focus
4. **Be clear in your prompts** - Tell Cursor exactly what type you're building
5. **Start simple** - You can always add features later
6. **Your choice affects everything** - Pages, features, design, all flow from this decision
