## Step-by-Step: Downloading Cursor

### Step 1: Go to Cursor Website

**1. Open your web browser:**
- Chrome, Firefox, Safari, or Edge
- Any browser works

**2. Go to cursor.sh:**
- Type "cursor.sh" in address bar
- Or search "Cursor code editor"
- Click the official website

**3. You'll see the Cursor homepage:**
- Welcome page
- Download button
- Information about Cursor

### Step 2: Download Cursor

**1. Click "Download" button:**
- Usually prominent on homepage
- Big download button
- Click it

**2. Choose your operating system:**
- Cursor detects your OS automatically
- Or choose manually:
  - Windows
  - Mac (Intel)
  - Mac (Apple Silicon)
  - Linux

**3. Download starts:**
- File downloads to your computer
- Usually goes to Downloads folder
- File name: `Cursor-Setup-x.x.x.exe` (Windows) or `Cursor-x.x.x.dmg` (Mac)

**4. Wait for download:**
- File size is about 100-200MB
- Takes a few minutes depending on internet speed
- Don't close browser

**Download complete!**

---

## Step-by-Step: Installing Cursor

### For Windows

**1. Find the downloaded file:**
- Go to Downloads folder
- Look for `Cursor-Setup-x.x.x.exe`
- Double-click to open

**2. Run the installer:**
- Windows may ask for permission
- Click "Yes" or "Run"
- Installer window opens

**3. Follow installation wizard:**
- Click "Next" to start
- Accept license agreement
- Choose installation location (default is fine)
- Choose start menu folder (default is fine)
- Click "Install"

**4. Wait for installation:**
- Progress bar shows installation
- Takes 1-2 minutes
- Don't close the window

**5. Complete installation:**
- Click "Finish" when done
- Cursor may launch automatically
- Or find it in Start menu

**Cursor is installed!**

### For Mac

**1. Find the downloaded file:**
- Go to Downloads folder
- Look for `Cursor-x.x.x.dmg`
- Double-click to open

**2. Open the disk image:**
- DMG file opens
- Shows Cursor application
- May show installation instructions

**3. Drag Cursor to Applications:**
- Drag Cursor icon to Applications folder
- Or follow on-screen instructions
- This installs Cursor

**4. Eject the disk image:**
- Right-click DMG in Finder
- Click "Eject"
- Can delete DMG file

**5. Open Cursor:**
- Go to Applications folder
- Double-click Cursor
- Mac may ask for permission
- Click "Open"

**Cursor is installed!**

### For Linux

**1. Find the downloaded file:**
- Usually a `.AppImage` or `.deb` file
- In Downloads folder

**2. For .deb files (Debian/Ubuntu):**
- Open terminal
- Navigate to Downloads: `cd ~/Downloads`
- Install: `sudo dpkg -i cursor-*.deb`
- Fix dependencies if needed: `sudo apt-get install -f`

**3. For .AppImage files:**
- Make executable: `chmod +x Cursor-*.AppImage`
- Run: `./Cursor-*.AppImage`

**4. Create desktop shortcut (optional):**
- Can add to applications menu
- Or create launcher

**Cursor is installed!**

## Common Installation Issues

### Problem: "Installation blocked"

**Solution:**
- Windows: Check Windows Defender
- Mac: System Preferences → Security → Allow
- May need to right-click → Open (Mac)
- Grant permissions

### Problem: "Can't find Cursor after installation"

**Solution:**
- Windows: Check Start menu, search "Cursor"
- Mac: Check Applications folder
- Linux: Check applications menu
- May need to restart computer

### Problem: "Cursor won't open"

**Solution:**
- Check system requirements
- Try restarting computer
- Re-download installer
- Check for error messages
- Contact Cursor support if needed

### Problem: "Installation takes too long"

**Solution:**
- Normal installation takes 1-5 minutes
- Depends on computer speed
- Be patient, don't interrupt
- If stuck for 10+ minutes, try again
