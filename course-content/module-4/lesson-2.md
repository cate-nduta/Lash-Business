## Opening Cursor

### First Launch

**When you open Cursor:**
- Welcome screen may appear

**To open a project:**
- Click "File" ‚Üí "Open Folder"

## Keyboard Shortcuts

### Essential Shortcuts

**File Operations:**
- `Ctrl+N` / `Cmd+N` = New file
- `Ctrl+O` / `Cmd+O` = Open file
- `Ctrl+S` / `Cmd+S` = Save
- `Ctrl+W` / `Cmd+W` = Close tab

**Navigation:**
- `Ctrl+P` / `Cmd+P` = Quick open file
- `Ctrl+B` / `Cmd+B` = Toggle sidebar
- `Ctrl+` ` / `Cmd+` ` = Toggle terminal

**AI Features:**
- `Ctrl+L` / `Cmd+L` = Open AI chat
- `Ctrl+I` / `Cmd+I` = Open Composer
- `Tab` = Accept AI suggestion

**Editing:**
- `Ctrl+F` / `Cmd+F` = Find
- `Ctrl+H` / `Cmd+H` = Replace
- `Ctrl+/` / `Cmd+/` = Toggle comment

**Learn shortcuts as you go!**

## Common Questions

### "How do I open a folder?"

**Answer:** File ‚Üí Open Folder, or `Ctrl+K Ctrl+O`, or drag folder into Cursor.

### "How do I close the sidebar?"

**Answer:** Click the sidebar icon, or `Ctrl+B` / `Cmd+B`, or View ‚Üí Appearance ‚Üí Primary Side Bar.

### "Where is the terminal?"

**Answer:** Bottom panel, click "Terminal" tab, or press `Ctrl+` ` / `Cmd+` `.

### "How do I search in files?"

**Answer:** Click search icon in sidebar, or `Ctrl+Shift+F` / `Cmd+Shift+F`.

### "How do I change theme?"

**Answer:** `Ctrl+K Ctrl+T` / `Cmd+K Cmd+T`, or File ‚Üí Preferences ‚Üí Theme.
