# Lesson 3.2: Creating a GitHub Account

**Estimated Time**: 15 minutes

---

## Introduction

Before you can use GitHub to save your code, you need an account. This lesson walks you through creating a GitHub account step by step. It's free, simple, and takes just a few minutes.

**What You'll Learn:**
- How to create a GitHub account
- What information you need
- How to verify your email
- How to set up your profile
- What to do after creating your account

---

## Before You Start

### What You'll Need

**1. Email Address:**
- Any email address works
- Use one you check regularly
- Will be used for notifications
- Can be changed later

**2. Username:**
- Choose a username
- Will be part of your GitHub URL
- Should be professional
- Can't be changed easily (choose carefully)

**3. Password:**
- Strong password
- At least 8 characters
- Mix of letters, numbers, symbols
- Keep it secure

**That's all you need!**

---

## Step-by-Step: Creating Your Account

### Step 1: Go to GitHub

**1. Open your web browser**
- Chrome, Firefox, Safari, or Edge
- Any browser works

**2. Go to github.com**
- Type "github.com" in address bar
- Or search "GitHub" and click first result

**3. You'll see the GitHub homepage**
- Welcome page with sign-up form
- Ready to create account

### Step 2: Fill Out the Sign-Up Form

**On the GitHub homepage, you'll see:**

**Username field:**
- Choose a username
- Examples: "yourname", "yourbusiness", "yourname-dev"
- Must be unique (GitHub will tell you if taken)
- Can contain letters, numbers, hyphens
- No spaces allowed

**Email field:**
- Enter your email address
- Use a real email you can access
- You'll need to verify it

**Password field:**
- Create a strong password
- At least 8 characters
- Mix of uppercase, lowercase, numbers
- GitHub will show strength indicator

**Tips for choosing username:**
- Use your real name or business name
- Keep it professional
- Make it memorable
- Avoid numbers if possible
- Check if it's available

### Step 3: Verify You're Human

**GitHub will ask you to verify:**
- Usually a simple puzzle
- Click the right images
- Or solve a simple challenge
- Takes just a few seconds

**This prevents spam accounts.**

### Step 4: Choose Your Plan

**GitHub will ask about your plan:**

**Free Plan (Recommended):**
- Click "Continue for free"
- Perfect for your needs
- Unlimited public repositories
- Unlimited private repositories (limited collaborators)
- Everything you need

**Pro Plan:**
- Paid option
- Not needed for this course
- Skip this for now

**Choose the free plan!**

### Step 5: Customize Your Experience (Optional)

**GitHub may ask questions:**
- What brings you to GitHub?
- Your experience level?
- What you want to build?

**You can:**
- Answer honestly (helps GitHub help you)
- Or skip these questions
- Doesn't affect your account

**Either way is fine!**

### Step 6: Verify Your Email

**1. Check your email:**
- GitHub sends verification email
- Check your inbox (and spam folder)
- Look for email from GitHub

**2. Click the verification link:**
- Opens in browser
- Confirms your email
- Account is verified

**3. If you don't see the email:**
- Check spam/junk folder
- Wait a few minutes
- Can resend from GitHub
- Check email address is correct

**Important:** You must verify your email to use GitHub fully!

## Setting Up Your Profile

### After Creating Your Account

**You'll be on your GitHub dashboard:**
- Welcome screen
- Getting started guide
- Can explore or set up profile

### Basic Profile Setup

**1. Add a Profile Picture (Optional but Recommended):**
- Click your profile icon (top right)
- Click "Settings"
- Click "Profile" in left sidebar
- Upload a photo
- Makes your profile look professional

**2. Add a Bio (Optional):**
- Short description of yourself
- Example: "Building a booking website"
- Can be changed anytime

**3. Add Your Location (Optional):**
- City, country
- Helps others find you
- Optional but nice to have

**You can skip these for now and do them later!**

## Understanding Your GitHub Account

### Your Profile URL

**Your GitHub profile will be:**
- github.com/yourusername
- Example: github.com/johndoe
- This is your public profile
- Others can see your repositories here

### Your Dashboard

**After logging in, you'll see:**
- Your dashboard
- Recent activity
- Your repositories
- Navigation menu

**This is your GitHub home!**

---

## Important Settings to Check

### Email Settings

**1. Go to Settings:**
- Click your profile icon (top right)
- Click "Settings"

**2. Check Email Settings:**
- Make sure email is verified
- Add backup email (optional)
- Set notification preferences

**3. Notification Settings:**
- Choose what emails you want
- Can adjust later
- Start with defaults

### Security Settings

**1. Two-Factor Authentication (Recommended):**
- Extra security layer
- Protects your account
- Can set up later
- Not required immediately

**2. Password:**
- Make sure it's strong
- Can change anytime
- Keep it secure

## Common Questions

### "What if my username is taken?"

**Answer:** Try variations:
- Add numbers: yourname123
- Add hyphens: your-name
- Add words: yourname-dev
- Be creative!

### "Can I change my username later?"

**Answer:** Yes, but it's complicated. Your old username becomes available to others, and links break. Better to choose carefully now.

### "Do I need to verify my email?"

**Answer:** Yes! You need to verify your email to:
- Create repositories
- Push code
- Use GitHub fully
- It's quick and easy

### "What if I don't receive the verification email?"

**Answer:**
- Check spam folder
- Wait a few minutes
- Check email address is correct
- Can resend from GitHub
- Contact GitHub support if needed

### "Is the free plan really free?"

**Answer:** Yes! The free plan is completely free forever. You get:
- Unlimited public repositories
- Unlimited private repositories
- All essential features
- Perfect for your needs
